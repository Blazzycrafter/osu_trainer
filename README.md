# osu_trainer

Modulare Version deines Training-Managers.

## Struktur
- `main.py`: Einstieg & Menüs
- `settings.json`: Konfiguration
- `core/`: System/Paths, Logger, Backup/Restore
- `modules/`: Osu-Manager, Template-Manager, API-Stub

## Ausführen
1. `pip install requests`
2. `python main.py`

> Hinweis: Nur Windows (nutzt `%APPDATA%`).
